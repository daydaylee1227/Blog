## Axios是什么

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中使用。



## Axios 功能

也是它的一些特点，有以下几点👇

- 基于promise的异步ajax请求库
- 浏览器/node端都可以使用
- 支持请求/响应式拦截
- 支持请求取消
- 自动转换JSON数据
- 转换请求数据和响应数据
- 客户端支持防御XSRF
